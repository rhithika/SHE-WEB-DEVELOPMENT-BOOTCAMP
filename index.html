HTML

HTML stands for HyperText Markup Language and is the language of the World Wide Web. It is the standard text formatting language used for creating and displaying pages on the Web.

CSS

CSS stands for Cascading Style Sheets and is the language for describing the presentation of Web pages, including colours, layout, and fonts, thus making our web pages presentable to the users.

JS

Javascript (JS) is a scripting languages, primarily used on the Web. It is used to enhance HTML pages and is commonly found embedded in HTML code.
